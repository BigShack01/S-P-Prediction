# NextDay-Stock-Prediction
Analysis with prediction of GIVEN stock ticker (interchangeable). Will flag a "buy" "hold" "sell" rating for the following day. Rough draft

An extremely rough version of a stock predictor
  Notes:
    - Will only predict "buy", "hold", or "sell" for next day trading. Will not predict what actual movement amounts will be
    - Will only predict one stock at a time. 
    - Stock ticker must be manually input into "next day signal model" section at bottom
    - Will use file titled "ppo_spy_model.zip" to pull ai model. If a model is not found, one will be created, and training will begin
      - Training iterations can be altered, model itself can also be altered
